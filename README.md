## VK Link Shortener
Этот скрипт позволяет создавать сокращённые ссылки через VK API и получать статистику переходов по уже сокращённым ссылкам.

## Основные возможности
- Создание коротких ссылок вида vk.cc/xxxxx для любых URL  
- Проверка, является ли ссылка уже сокращённой через VK  
- Получение статистики переходов по сокращённым ссылкам  

## Установка
1. Клонируйте репозиторий или скачайте файлы проекта  
2. Установите зависимости:  
````bash  
pip install -r requirements.txt
````
## Настройка
1. Получите токен доступа VK API:  
2. Перейдите на dev.vk.com  
3. Создайте новое приложение
4. Получите ключ доступа (access token)
5. Создайте файл .env в корневой директории проекта и добавьте токен:
````env
VK_API_TOKEN=ваш_токен_здесь
````
## Использование
- Через командную строку:  
````bash
python API_L2.py https://example.com
````

## Примеры работы
- Сокращение ссылки (если ссылка не является сокращённой VK):
````bash
python API_L2.py https://dvmn.org
# Вывод: Сокращенная ссылка: https://vk.cc/xxxxx
````
- Получение статистики (если ссылка уже сокращена через VK):
````bash
python API_L2.py https://vk.cc/xxxxx
# Вывод: Количество переходов по ссылке: 150
````
## Файлы проекта
1. API_L2.py - основной скрипт   
2. requirements.txt - зависимости Python  
3. .env - файл с токеном (создаётся пользователем)  
4. README.md - эта документация  

## Принцип работы
Скрипт проверяет, является ли введённая ссылка сокращённой через VK (проверяет домен vk.cc). Если ссылка обычная - создаёт короткую версию через VK API. Если ссылка уже сокращённая - получает статистику переходов
